About

  • This website aims to make it easier to calculate golf scores by allowing users to add a course to the database and then using it to calculate their scores. To caalculate their scores a player needs to input their handicap, scores and the course at which the round was played. To add a course into the database a player needs to enter the name, location, stroke index and par of the course.

    Currently available formats:

    1. Gross Score

    2. Nett Score

    3. Stableford Score

    4. Stableford Points Score

    5. Quota Score

    For information on these formats please see: http://golfcollege.edu/popular-golf-tournament-formats/
